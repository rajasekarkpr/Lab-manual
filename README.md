# VLSI Design -- ASIC Flow using Open Source Tool

A complete step-by-step manual covering:
- Linux (WSL2) environment setup on Windows
- Installing the open-source EDA toolchain (Docker, GHDL, OpenLane, KLayout, Git)
- VHDL-to-Verilog conversion
- The OpenLane RTL-to-GDSII ASIC implementation flow
- Viewing synthesis/signoff reports and chip layouts
- Publishing results via Git/GitHub

See [`VLSI_ASIC_OpenSource_Manual.pdf`](VLSI_ASIC_OpenSource_Manual.pdf) for the full manual.
